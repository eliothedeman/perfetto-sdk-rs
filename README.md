# perfetto-sdk-rs

[![Build Status](https://github.com/eliothedeman/perfetto-sdk-rs/workflows/Rust/badge.svg)](https://github.com/eliothedeman/perfetto-sdk-rs/actions)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Utilities for creating and working with perfetto traces in Rust.

## Crates

This workspace contains the following crates:

### perfetto-writer

[![Crates.io](https://img.shields.io/crates/v/perfetto-writer.svg)](https://crates.io/crates/perfetto-writer)
[![Documentation](https://docs.rs/perfetto-writer/badge.svg)](https://docs.rs/perfetto-writer)

A utility package for writing protobuf encoded perfetto traces.

### tracing-perfetto-writer

[![Crates.io](https://img.shields.io/crates/v/tracing-perfetto-writer.svg)](https://crates.io/crates/tracing-perfetto-writer)
[![Documentation](https://docs.rs/tracing-perfetto-writer/badge.svg)](https://docs.rs/tracing-perfetto-writer)

A `tracing-subscriber` Layer for writing protobuf encoded perfetto traces.
